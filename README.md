# Preview Editor

## Start Development

```bash
pnpm
pnpm start # start debug server
```

## Build

```bash
pnpm run build
pnpm run copy
```

### Release

```bash
pnpm run deploy # default release: github pages
```

### Run server

```bash
pnpm run serve
```

## Code Format

```bash
pnpm run format
```
